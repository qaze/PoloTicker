# PoloTicker

## Prerequisites

- Xcode 10.0+
- Cocoapods
- Poloniex Secret And API keys ( can be found inside your poloniex account )

## Basic setup

- Clone repository
- Run:

```
pod install
```
- Open PoloTicker.xcworkspace
- Put your Secret and API keys in PoloConnection.swift file
- Build & Run
